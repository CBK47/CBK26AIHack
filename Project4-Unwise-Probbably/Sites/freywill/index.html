FREYWILL

Pay-per-call AI. 120B local inference. 90% below market.

Project Brief

One-Liner

FREYWILL is a pay-per-call AI marketplace: "Fiverr for AI agents" using local 120B inference.

Opportunity

Hackathon teams burn API credits fast; this stack uses GB10 local inference to cut market pricing by 80-90%.

Core Offer + Pricing

  • Summarize: 0.001 USDC
  • Complete: 0.001 USDC
  • Review: 0.005 USDC
  • Payment rail: x402 on Base/USDC, accountless pay-per-call flow.

Architecture

  • Flask API + x402 payment middleware + Ollama local models.
  • Models: gpt-oss:120b and qwen2.5-coder:32b.
  • Service discovery via /.well-known/x402 for autonomous agents.

Status + Roadmap

  • Scaffold and dashboard are done; API/payment lanes are in active build/mock mode.
  • Next commits: core API services, payment middleware, landing/deployment polish.
  • Project 4 ports: 4000-4007 active, 4008-4014 reserved.
